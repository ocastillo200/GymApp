## Instrucciones de uso
### Clonar repositorio
```bash
git clone https://github.com/ocastillo200/ProyectoIS.git
```

### Crear virtual environment
```bash
cd fastapi_mongo_project
python -m venv venv
./venv/Scripts/activate
```

### Instalar requirements
```bash
pip install -r requirements.txt
```
